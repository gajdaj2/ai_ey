# Lab 01 — LangChain + OpenAI

Ten katalog zawiera materiały w dwóch ścieżkach:

## Ścieżka A — Python od zera

- `tutorial-python.md` — podstawy składni, funkcje, OOP i praca z plikami,
- `cwiczenia-python/` — ćwiczenia krok po kroku do fundamentów Pythona.

Rekomendowana kolejność:

1. przeczytaj `tutorial-python.md`,
2. wykonaj ćwiczenia z `cwiczenia-python/`,
3. przejdź do ścieżki B.

## Ścieżka B — LangChain + OpenAI

Ta ścieżka zakłada podstawową znajomość Pythona.

- `tutorial.md` — krótkie wprowadzenie krok po kroku,
- `cwiczenia.md` — zestaw ćwiczeń praktycznych do samodzielnego wykonania.

Rekomendowana kolejność pracy:

1. przeczytaj `tutorial.md`,
2. uruchom przykłady we własnym pliku `.py`,
3. wykonaj ćwiczenia 1–4,
4. przejdź do zadań rozszerzonych.

Do pracy potrzebujesz aktywnego `OPENAI_API_KEY` oraz pakietów z głównego `requirements.txt`.
